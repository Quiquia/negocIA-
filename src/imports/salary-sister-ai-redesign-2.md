RESPONSIVE DESIGN REQUIREMENT

The entire product must be fully responsive.

Design for three breakpoints:

Desktop
Tablet
Mobile

The layout must adapt fluidly.

On mobile devices, prioritize vertical layouts, stacked components, and thumb-friendly interactions.

VISUAL STYLE DIRECTION

Avoid minimalist design with large empty spaces.

The interface should include:

• gradient backgrounds
• colorful sections
• visual cards
• icons and illustrations
• layered components
• stronger typography hierarchy

The layout should feel dense enough to communicate value but still organized and readable.

Use color to guide attention and emphasize insights.
