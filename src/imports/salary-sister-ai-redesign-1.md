Guest mode

• run a salary analysis
• view results
• try the negotiation simulator

“Guarda tu análisis creando una cuenta gratuita.”

Registered user mode

Users can:

• save salary analysis history

• track career progress

“Crear cuenta y guardar mi análisis”
